# HAY (How Are You?) - server
📞 통화 상대의 스케줄 확인이 가능한 전화 앱
## 서비스 설명
- 전화를 걸었는데 상대방이 받지 않아 불편했던 경험 📵
- 일하는 중이라 걸려온 전화를 못 받아서 불편했던 경험 📵

을 토대로 전화를 걸기 전, **상대의 통화가능 여부(스케줄)** 을 미리 알 수 있는 전화앱을 기획했습니다.

### 기능
- 전화 기능
- 전화번호부 기능
- 소셜 로그인 기능
- 일주일간 스케줄 입력,조회,삭제,수정 기능
- 현재 상태글 설정 기능
- 사용자별 숨김 기능
